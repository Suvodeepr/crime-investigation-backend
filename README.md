# 🚔 AI Crime Investigation System (MERN + Groq AI)


![CRIME_INVASTIGATION_SYSTEM](CRIME_INVASTIGATION_SYSTEM_IMAGE.png)

This is an AI-based Crime Investigation System built using the MERN stack.

The system helps police and authorities to:
- Manage crime records
- Track cases
- Store evidence
- Analyze data using AI

---

# 📌 Introduction

This project uses modern web technologies and AI to improve the crime investigation process.

It makes the system:
- Faster
- Smarter
- More organized

---

# 🎯 Objectives

- Automate crime investigation work
- Use AI to identify suspects faster
- Track cases in real-time
- Improve analysis using AI
- Build a scalable cloud system

## 2️⃣ Frontend

# 🔗 Example API

### FIR APIs

- GET all FIRs → `/fir`
- POST FIR → `/fir`
- GET single FIR → `/fir/:id`
- UPDATE FIR → `/fir/:id`
- DELETE FIR → `/fir/:id`
